# State

Each store that you implement in the project must extend **ObjectLiteral** (key value object) in order to manage its state changes optimally.

```js  {3} copy
let a = 1;

console.log(a);
```

## State

import { useState } from 'react'

{/* Import CSS modules */}
import styles from '../components/counters.module.css'

export const Counter = () => {
  const [count, setCount] = useState(0);
  return (
    <div>
      <button onClick={() => setCount(count + 1)} className={styles.counter}>Clicked {count} times</button>
    </div>
  );
};

<Counter/>
